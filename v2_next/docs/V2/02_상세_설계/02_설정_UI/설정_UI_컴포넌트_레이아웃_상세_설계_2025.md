# 설정 UI 컴포넌트/레이아웃 상세 설계 (2025)

## 1. 화면 구조 개요
- 목표: 설정 변경을 빠르게 찾고, 안전하게 저장하며, 영향 범위를 즉시 이해하도록 구성한다.
- 레이아웃: 좌측 섹션 탐색 + 우측 상세 폼 + 상단 상태 바 + 하단 액션 바.

```
[상단 상태 바]
[좌측 섹션 목록] | [우측 상세 폼 영역]
[하단 액션 바]
```

## 2. 상단 상태 바 구성
- 설정 파일 경로 표시: 단일 라인, 길면 말줄임.
- 변경 상태 배지: 변경 있음/없음, 변경 카운트.
- 재시작 필요 배지: 저장 후 지속 표시.
- 연결 상태 요약(선택): 현재 통신 OK/Warning/Offline.

### 컴포넌트
- 상태 배지(중립/경고/위험 3단)
- 경로 복사 버튼(클립보드)
- 경로 오류 아이콘(권한/읽기 전용 감지 시)

## 3. 좌측 섹션 탐색(네비게이션)
- 섹션: 요약, 통신/장비, SPOT 카메라, 저장/로깅, 알림/임계값, 보안/권한, 고급.
- 현재 섹션 하이라이트.
- 변경이 있는 섹션에 점 배지 표시.
- 좁은 화면에서는 상단 드롭다운으로 축소.

## 4. 우측 상세 폼(섹션별)
### 4.1 요약(Overview)
- 핵심 설정 요약 카드 3개
  - 통신 요약(IP/Port)
  - 저장 요약(로그/스냅샷 경로)
  - SPOT 요약(IP/Refresh)
- 최근 저장 시각 + 백업(.bak) 상태

### 4.2 통신/장비
- 입력: Extruder IP/Port, LS PLC IP/Port
- 액션: 개별 연결 테스트 버튼
- 결과: 성공/실패 배지 + 마지막 테스트 시각

### 4.3 SPOT 카메라
- 입력: SPOT IP, Refresh Interval
- 미리보기: 이미지 URL 샘플 표시(읽기 전용)
- 상태: 최근 수신 시간/지연 상태 배지

### 4.4 저장/로깅
- 입력: 로그 경로, 스냅샷 경로
- 스위치: 자동 저장
- 선택: 회전 모드(DAILY/BILLET)
- 경로 검증: 쓰기 가능/권한/존재 여부 표시
- NAS 배지: Z: 드라이브 감지 시 안내

### 4.5 알림/임계값
- 토글: 알림 활성/비활성
- 입력: 임계값 범위(속도/압력/온도 등)
- 주의: 범위 밖 입력 시 즉시 경고 라벨

### 4.6 보안/권한
- 입력: 설정 비밀번호
- 상태: 설정됨/미설정 표시
- 안내: 읽기 전용 파일 시 관리자 권한 필요 메시지

### 4.7 고급
- 환경변수 우선 적용 토글
- 인코딩 정책 안내(UTF-8 BOM)
- 설정 내보내기/가져오기(선택 기능 자리)

## 5. 하단 액션 바
- 좌측: “변경 사항 N개” 표시
- 우측: 저장 / 취소 / 기본값 복원 / 백업 복원
- 저장 버튼은 유효성 검사 통과 시에만 활성화

## 6. 입력 컴포넌트 규격
- IP 입력: IPv4 형식 유효성 검사
- Port 입력: 숫자 범위(1~65535)
- 숫자 입력: step 표기, 최소/최대 안내
- 경로 입력: 폴더 선택 버튼(가능 시)
- 토글 스위치: ON/OFF 텍스트 병기

## 7. 상태/피드백 규칙
- 저장 성공: 상단 배지 + 토스트 + 재시작 필요 표기 유지
- 저장 실패: 상단 오류 배너 + 필드 강조
- 테스트 중: 로딩 스피너 + 버튼 비활성화
- 읽기 전용 감지: 저장 버튼 비활성화 + 해결 안내

## 8. 마이크로카피(권장 문구)
- “변경 사항은 재시작 후 적용됩니다.”
- “경로에 쓰기 권한이 없습니다. 관리자 권한을 확인하세요.”
- “연결 테스트를 실행하면 현재 설정으로 즉시 검증합니다.”
- “백업을 복원하면 현재 설정이 덮어써집니다.”

## 9. 디자인/스타일 가이드
- 폼 필드는 카드 안에 그룹화하고 그룹 제목을 명확히 한다.
- 버튼은 주/보조/위험(복원)으로 구분.
- 숫자 표시 영역은 tabular nums로 폭 고정.
- 경고 배지는 노랑, 오류는 붉은색, 정상은 녹색.

## 10. 구현 체크리스트
- 섹션 전환 시 스크롤 위치가 안정적으로 이동하는가.
- 변경 카운트가 정확히 갱신되는가.
- 저장 실패 시 어떤 항목이 문제인지 즉시 식별되는가.
- NAS 경로(Z:) 사용 시 안내가 표시되는가.
- 재시작 필요 배지가 저장 이후 유지되는가.
