# 설정 UI 디자인 설계 (2025)

## 1. 목적 및 범위
- 현장 운영자가 설정 변경을 빠르고 안전하게 수행하도록 돕는다.
- 변경 사항의 영향 범위(즉시/재시작 필요)를 명확히 보여준다.
- 설정 파일(config.ini)을 기반으로 이식 PC에서도 동일한 UX를 제공한다.

## 2. 핵심 사용자 시나리오
- 현장 엔지니어: 장비 IP/Port 변경 후 연결 테스트 → 즉시 상태 확인.
- 운영 담당: 로그/스냅샷 저장 경로 변경, NAS 경로 유지.
- 관리자: 비밀번호 설정, 변경 이력 확인, 설정 내보내기.
- 유지보수: 설정 백업/복원, 파일 오류 복구.

## 3. 정보 구조(IA) 제안
- 요약: 현재 설정 파일 경로, 마지막 저장 시각, 변경 미적용 배지, 재시작 필요 표시.
- 통신/장비: Extruder/LS PLC IP/Port, 연결 테스트, 네트워크 상태.
- SPOT 카메라: IP, Refresh Interval, 이미지 URL 미리보기.
- 저장/로깅: 로그 경로, 스냅샷 경로, 자동 저장, 회전 정책.
- 알림/임계값: 알림 on/off, 임계값 범위, 경고 유지 시간.
- 보안/권한: 설정 비밀번호, 읽기 전용 상태, 관리자 안내.
- 고급: 환경변수 우선 적용 여부, 인코딩/백업 관리.

## 4. 레이아웃 및 화면 구성
- 좌측 메뉴 + 우측 상세 패널 구조로 섹션 이동을 명확히 한다.
- 상단 고정 바: 설정 파일 경로 + 적용 상태 + 재시작 필요 배지.
- 하단 고정 액션: 저장, 취소, 기본값 복원, 백업/복원.
- 좁은 화면(현장 PC) 대비: 아코디언 방식으로 축소 가능.

## 5. 필수 기능(2025 실무 기준)
1) 변경 사항 추적
- 변경된 항목 하이라이트, 변경 카운트 표시.
- 저장 전 확인 다이얼로그(요약 + 영향 범위).

2) 연결 테스트
- 장비별 테스트 버튼 제공.
- 테스트 결과 배지(성공/실패/지연) + 최근 테스트 시각.

3) 경로 검증 및 NAS 친화성
- 경로 존재/권한/쓰기 가능 여부 확인.
- NAS 경로(Z:) 사용 시 안내 배지.

4) 백업/복원
- 저장 시 자동 백업(.bak) 표시.
- 수동 백업/복원 버튼 + 복원 시 확인 메시지.

5) 재시작 필요 표시
- 저장 완료 후 배지 유지.
- 재시작 버튼 또는 안내 문구 제공.

## 6. 선택 기능(권장)
- 설정 내보내기/가져오기: 다른 PC로 이식 시 시간 절약.
- 변경 이력 로그: 누가 언제 무엇을 변경했는지 기록.
- 설정 프로파일: 라인/제품별 템플릿 관리.
- 읽기 전용 감지: PermissionError 시 즉시 안내.

## 7. 입력 검증 및 안전 장치
- IP/Port 형식 검증 및 범위 제한.
- 수집 주기/로그 회전 값: 범위 안내 + 잘못된 값 차단.
- 저장 전 실시간 검증 메시지 제공.
- 잘못된 입력은 해당 필드 강조 + 구체적 오류 메시지.

## 8. 상태 피드백 설계
- 저장 성공/실패 토스트 + 상세 메시지.
- 연결 테스트 중 로딩 표시, 완료 후 결과 배지.
- 마지막 성공 시각은 모든 항목에 통일된 패턴 적용.

## 9. 현장 사용성(Industrial UX) 기준
- 큰 터치 영역(최소 44px), 굵은 대비 색상.
- 숫자 폭 고정(tabular nums)으로 흔들림 방지.
- 읽기 쉬운 마이크로카피: 재시작 후 적용됩니다.
- 오류 시 해결 가이드를 간단히 제시.

## 10. 검수 체크리스트
- 저장 시 config.ini 백업이 생성되는가.
- 재시작이 필요한 항목이 정확히 표시되는가.
- NAS 경로(Z:) 사용 시 권한/연결 오류를 안내하는가.
- 연결 테스트 결과가 즉시 반영되는가.
- 입력 검증 실패 시 저장이 비활성화되는가.
