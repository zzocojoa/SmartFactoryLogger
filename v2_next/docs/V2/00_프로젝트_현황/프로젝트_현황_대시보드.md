## 3. 누락 및 미구현 항목 리포트 (Gap Analysis)

### 3.1 설계 문서는 있으나 구현 내용이 없는 기능
- 중앙 설정 서비스/하이브리드 캐시/롤백(서버·DB 미구현)
- 로컬 오버라이드 UI/RBAC(권한 정책 미구현)
- 동기화 에이전트(구현 부재)
- 패키징/Electron 빌드 및 설치 자동화(파이프라인 미구현)
- 경로 검증 API 및 NAS 상태 체크(구현 여부 미확인)
- 임계값(thresholds) 백엔드 적용 로직(미구현)

### 3.2 구현은 되었으나 검증(테스트) 문서가 없는 기능
- config.ini 핫 리로드 적용/보류 로직 테스트 시나리오 부재
- REAL 통신 드라이버(V1 대비 정합성) 검증 문서 부재
- Grafana Scenes 개선 1~5단계 회귀 테스트 문서 부재
- 레이아웃 스냅/충돌 방지 검증 기록 부재

### 3.3 문서 불일치/중단/인코딩 문제
- 인코딩 깨짐 문서가 다수 존재(UTF-8 재저장 필요)

  - 운영/배포 문서: 배포_전략, 중앙_설정_*
    로컬_오버라이드, 동기화_에이전트
  - 구현 결과/코드 리뷰: 현재_구현_현황, 그라파나_씬_개선
    코드_검토_및_개선안, 그라파나_씬_엔진_코드_검토
