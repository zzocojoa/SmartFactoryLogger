# 설정 오버라이드 권한 정리

> **문서 작성일**: 2026-01-08\
> **대상 버전**: V2

## 개요

V2 설정 창(Menu → 설정)에서는 **오버라이드 ON/OFF** 상태에 따라 변경 가능한
항목이 구분됩니다.\
오버라이드는 설정 변경 권한을 제어하는 보안 기능으로, 활성화 시 관리자 비밀번호
인증이 필요합니다.

---

## 🔐 오버라이드 ON 필요 항목

> 아래 항목들은 **오버라이드 ON** 상태에서만 저장 가능합니다.\
> 오버라이드가 OFF면 값을 수정할 수는 있으나, **저장 버튼이 비활성화**됩니다.

| 카테고리                | 항목                                                       |
| :---------------------- | :--------------------------------------------------------- |
| **압출기 (Extruder)**   | IP 주소, 포트                                              |
| **LS PLC**              | IP 주소, 포트                                              |
| **SPOT 카메라**         | IP 주소, 새로고침 간격                                     |
| **임계값 (Thresholds)** | 마스터 스위치, 각 센서별 임계값 ON/OFF 및 수치             |
| **저장 경로**           | 로그 경로, 스냅샷 경로, 자동 저장 여부                     |
| **로테이션 설정**       | 로테이션 활성화, 모드(DAILY/BILLET), 유휴시간, 압력 임계값 |
| **보안**                | 관리자 비밀번호 변경                                       |

### 오버라이드 ON 필요 액션 버튼

- `저장`
- `기본값 복원`
- `백업 복원`

---

## ✅ 오버라이드 상관없이 항상 가능

> 아래 항목들은 오버라이드 상태와 관계없이 조회 또는 실행할 수 있습니다.

| 카테고리           | 항목                                       |
| :----------------- | :----------------------------------------- |
| **설정 요약 조회** | 통신 요약, 저장 요약, SPOT 요약 카드       |
| **적용 상태 조회** | 즉시 적용 항목 목록, 재시작 필요 항목 목록 |
| **메타 정보 조회** | 버전, 마지막 동기화 시간, 소스             |
| **Central 상태**   | 중앙 서버 연결 상태 조회                   |
| **액션**           | 닫기 버튼, 설정 창 열기/닫기               |

---

## 🔄 오버라이드 토글

- **"오버라이드 켜기/끄기" 버튼**은 오버라이드 상태와 관계없이 **항상 누를 수
  있음**
- 단, 토글 시 **관리자 비밀번호 인증**이 필요함

---

## 요약

| 작업 유형           | 오버라이드 필요 여부    |
| :------------------ | :---------------------- |
| 설정 **조회**       | ❌ 불필요               |
| 설정 **저장/복원**  | ✅ 필요 (비밀번호 인증) |
| 오버라이드 **토글** | ✅ 비밀번호 인증 필요   |

---

## ⚡ 즉시 적용 vs 재시작 필요

> 설정을 저장하면 일부 항목은 **즉시 적용**되고, 일부는 **백엔드 재시작 후
> 적용**됩니다.

### ✅ 즉시 적용 항목 (Hot Reload)

| 카테고리        | 항목                                                                                    |
| :-------------- | :-------------------------------------------------------------------------------------- |
| **로깅 설정**   | Log Path, 자동 저장, 로테이션 활성화/모드, 유휴시간, 압력 임계값, CSV 헤더              |
| **스냅샷**      | Snapshot Path                                                                           |
| **SPOT 카메라** | IP, 이미지 URL, 새로고침 간격, 크로스헤어 설정, 포커스 설정, 액추에이터 설정, 위젯 크기 |
| **PLC 통신**    | Extruder IP/Port, LS PLC IP/Port/Targets                                                |
| **임계값**      | 마스터 스위치, 각 센서별 임계값 ON/OFF 및 수치                                          |
| **보안**        | 비밀번호 설정                                                                           |
| **시스템**      | 폴링 간격 (interval_sec)                                                                |

---

### 🔄 재시작 필요 항목

| 상황               | 설명                                          |
| :----------------- | :-------------------------------------------- |
| **PLC 연결 실패**  | Extruder/LS PLC 설정 변경 후 연결 실패 시     |
| **적용 예외 발생** | 설정 적용 중 오류 발생 시 pending 상태로 전환 |

> 💡 **참고**: 대부분의 설정은 즉시 적용되지만, **PLC 연결 설정**은 연결
> 재시도에 실패하면 재시작이 필요할 수 있습니다.

---

### 프론트엔드 표시

설정 창의 "적용 상태" 섹션에서 다음을 확인할 수 있습니다:

- **즉시 적용됨**: 실시간으로 반영된 항목 목록
- **재시작 필요**: 재시작 후 적용될 항목 목록
